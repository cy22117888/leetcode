169、多数元素
===
给定一个大小为 n 的数组，找到其中的多数元素。多数元素是指在数组中出现次数大于 ⌊ n/2 ⌋ 的元素。<br>
你可以假设数组是非空的，并且给定的数组总是存在多数元素。<br>

示例 1:<br>
```
输入: [3,2,3]
输出: 3
```
示例 2:<br>
```
输入: [2,2,1,1,1,2,2]
输出: 2
```

```
public int majorityElement(int[] nums) {
    /**
     * 首先对属于进行排序。
     * 因多数元素的出现次数是大于一半的，所以有序数组中的多数元素一定是也只能是最中间位置的元素
     */ 
    Arrays.sort(nums);
    return nums[nums.length / 2];
}
```
