476、数字的补数
===

给定一个正整数，输出它的补数。补数是对该数的二进制表示取反。<br>

注意:<br>
* 给定的整数保证在32位带符号整数的范围内。
* 你可以假定二进制数不包含前导零位。

示例 1:<br>
```
输入: 5
输出: 2
解释: 5的二进制表示为101（没有前导零位），其补数为010。所以你需要输出2。
```
示例 2:<br>
```
输入: 1
输出: 0
解释: 1的二进制表示为1（没有前导零位），其补数为0。所以你需要输出0。
```

### 位运算
```
// 思路：原数值 + 补数 + 1 = 2的（总位数）次幂
public int findComplement(int num) {
    
    int arg = num;
    int sum = 1;

    while (arg != 0) {
        arg >>= 1;
        sum <<= 1;
    }

    return sum - 1 - num;
}
```
